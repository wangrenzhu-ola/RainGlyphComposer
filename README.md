# RainGlyph Composer

RainGlyph Composer is a native SwiftUI iOS app that turns a short rain-tap feeling into a saved visual glyph score. The app is intentionally local-first: user-created score cards, starter examples, and premium local style packs stay on device.

## Open in Xcode

Open `RainGlyphComposer.xcodeproj`, select the `RainGlyphComposer` scheme, and run on an iPhone simulator.

## Core flows

- Rain Score Shelf with a product-specific empty state and local starter disclosure.
- Rain Tap Setup for manual rain-tap capture plus optional system-assisted draft suggestions.
- Glyph Staff Composer for editing visual parameters, density beads, and glyph staff.
- Rain Glyph Score Detail for edit/favorite/delete confirmation.
- Premium Local Packs as a non-blocking StoreKit 2 entry for local visual/theme packs.
